# Nextgen Digital Leads

A premium technology website for Nextgen Digital Leads, built with HTML, CSS, and JavaScript.

## Features

- Sticky navigation bar with mobile menu
- Full-screen hero section with animated background and floating particles
- About, Services, Why Choose Us, Portfolio, Process, Pricing, Testimonials, FAQ, Contact
- Glassmorphism styling, gradient borders, green glow effects, and smooth reveal animations
- Animated counters, hover interactions, back-to-top button, and responsive layout
- Accessible and optimized for fast static deployment on Render

## Run Locally

Open `index.html` in a browser or serve the folder using a static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deploy on Render

This project is ready to deploy as a static site on Render. Create a new Static Site in Render and point it to this repository.

If you use `render.yaml`, Render can automatically detect the service configuration.
